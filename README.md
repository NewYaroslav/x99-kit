# x99-kit
Драйверы, утилиты и заметки по установке для плат X99/LGA2011-3

## Полезные ресурсы

* [e5450.com](https://e5450.com/) - русскоязычный ресурс про “вторичный рынок” железа, в первую очередь про Xeon и сборки на серверных/китайских платформах, с гайдами, тестами, модами BIOS и т.п.
* [Китайский LGA2011-3 (X99 Huananzhi, Jingsha, Aterminer, PLEXHD)](https://forums.overclockers.ru/viewtopic.php?f=1&start=45920&t=602922)
* [Intel Chipset Device Software](https://www.intel.com/content/www/us/en/download/19347/chipset-inf-utility.html) - Данная утилита версии 10.1.20398.8776 предназначенна в первую очередь для чипсетов Intel®, устанавливает INF-файлы Windows
* [Intel Management Engine Interface (MEI/AMT)](https://www.station-drivers.com/index.php?Itemid=352&func=fileinfo&id=2265&lang=en&option=com_remository) - Драйверы Intel® Management Engine для X99. Альтернативно скачать можно так же тут: [e5450.com](https://e5450.com/socket-2011-3/drivers-x99/)
* Драйверы [Realtek HDA/UAD/Component Drivers](https://www.station-drivers.com/index.php/en-us/forum/realtek-hda-uad-drivers-firmwares-utilities/24-realtek-hda-uah-component-drivers) с форума
